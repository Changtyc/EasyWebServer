# CGI-Learn

- 该文件夹下，实现的是一个CGI服务器。
- 实现的 CGI 服务器为：客户端连接后，发送想要打开的应用程序名称，后接换行符表示结束。客户端查看若该程序存在，则新建子进程运行该程序，否则重新接收。
- 基于高效的半同步/ 半异步模式





# GET-WebServer

- 该文件夹下，实现的是一个简单的web服务器，仅支持GET功能。
- 基于半同步半反应堆模式。





# POST-Webserver

- 该文件下，实现的是一个相对完整的web服务器，支持GET和POST功能。
- 在GET-WebServer项目的基础上进行改进的。
